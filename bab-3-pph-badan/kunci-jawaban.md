# KUNCI JAWABAN - BAB 3: PPh Badan

## Studi Kasus

### Kasus 1: PT Maju Sejahtera (Pasal 31E)
1. **Koreksi Positif:** 120jt + 75jt + 15jt + 25jt + 30jt = **Rp265.000.000**
   **Koreksi Negatif:** 50jt + 25jt = **Rp75.000.000**
2. **PKP Badan** = 17.290.000.000 + 265.000.000 - 75.000.000 = **Rp17.480.000.000**
3. **PPh terutang 31E:**
   - PKP s.d. Rp4,8M: Rp4.800.000.000 x 11% = Rp528.000.000
   - PKP > Rp4,8M: (Rp17.480.000.000 - Rp4.800.000.000) = Rp12.680.000.000 x 22% = Rp2.789.600.000
   - **Total PPh Terutang = Rp528.000.000 + Rp2.789.600.000 = Rp3.317.600.000**
4. **Jurnal:**
   - Beban PPh Badan (Debit) Rp3.317.600.000
   - Utang PPh Badan (Kredit) Rp3.317.600.000

### Kasus 2: Perusahaan Besar
1. Tarif 22% (perusahaan besar, omzet > Rp50M, tidak dapat fasilitas 31E)
2. PPh = 22% x Rp23.000.000.000 = **Rp5.060.000.000**
3. Kredit pajak = Rp50jt + Rp75jt = Rp125.000.000
   PPh kurang bayar = Rp5.060.000.000 - Rp125.000.000 = **Rp4.935.000.000**

### Kasus 3: UMKM
1. PPh Final = 0,5% x Rp4.800.000.000 = **Rp24.000.000**
2. Final: bayar Rp24jt, selesai, tidak bisa kredit. Normal: perlu hitung PKP, jika PKP kecil mungkin bayar lebih rendah.
3. **Tidak bisa dikreditkan.** PPh Final bersifat final.

## Soal Konseptual
1. **PPh Final** = pajak selesai saat dibayar, tidak bisa dikreditkan. **PPh Tidak Final** = pajak di muka, bisa dikreditkan di SPT Tahunan.
2. **World Wide Income** = penghasilan WP Badan DN diakui dari seluruh sumber, baik dalam maupun luar negeri.
3. **Koreksi Positif:** penyusutan komersial > fiskal, natura, sanksi pajak. **Koreksi Negatif:** pendapatan final, laba selisih kurs belum realisasi.
4. **Laba komersial** = laba versi akuntansi. **Laba fiskal** = laba setelah penyesuaian ketentuan pajak.
5. Kerugian fiskal dapat dikompensasi ke laba tahun berikutnya maksimal 5 tahun.
6. 22% (umum), 19% (Tbk 40% publik), 11% (31E s.d. Rp4,8M PKP), 0,5% (UMKM final).
7. Omzet <= Rp50M, PKP dihitung dengan pembukuan, bukan final.
8. PPh 22/23 adalah kredit pajak, mengurangi PPh Badan terutang di akhir tahun.

## Soal Aplikasi
1. PKP Rp1M < Rp4,8M: PPh = Rp1.000.000.000 x 11% = **Rp110.000.000**
2. Kurang bayar = Rp110.000.000 - Rp50.000.000 - Rp25.000.000 = **Rp35.000.000**
3. PKP Rp5M: Rp4,8M x 11% + Rp200jt x 22% = Rp528jt + Rp44jt = **Rp572.000.000**
4. a. Final: 0,5% x Rp4,5M = **Rp22.500.000**. b. Normal: misal PKP Rp500jt, PPh = Rp500jt x 11% = Rp55jt. Final lebih menguntungkan.
5. PKP = Rp2M + Rp300jt - Rp100jt = **Rp2.200.000.000**. PPh = Rp2,2M x 11% = **Rp242.000.000**.
