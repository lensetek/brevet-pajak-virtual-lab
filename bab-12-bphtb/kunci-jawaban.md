# KUNCI JAWABAN - BAB 12: BPHTB

## Studi Kasus

### Kasus 1: Jual Beli Rumah
1. NPOP = **Rp500.000.000** (dari `sspd-bphtb.pdf`)
2. NPOPTKP DKI Jakarta = **Rp80.000.000** (dari `tabel-npoptkp.pdf`)
3. NPOP Kena Pajak = Rp500.000.000 - Rp80.000.000 = **Rp420.000.000**
4. BPHTB = 5% x Rp420.000.000 = **Rp21.000.000**
5. Penjual: PT Graha Properti Prima. Pembeli: Budi Santoso (dari `ajb-kutipan.pdf`)
6. BPHTB harus dibayar sebelum AJB ditandatangani sebagai syarat balik nama sertifikat.

### Kasus 2: Warisan
1. NPOPTKP warisan = **Rp300.000.000** (lebih besar dari jual beli)
2. NPOP Kena Pajak = Rp1M - Rp300jt = Rp700jt. BPHTB = 5% x Rp700jt = **Rp35.000.000**
3. Warisan mendapat NPOPTKP lebih besar (Rp300jt vs Rp80jt), sehingga lebih ringan.

### Kasus 3: Perbedaan PBB vs BPHTB
- Waktu: PBB tahunan, BPHTB saat transaksi
- Dasar: PBB = NJOP, BPHTB = NPOP (harga transaksi)
- Tarif: PBB = 0,1-0,3%, BPHTB = 5%
- Sifat: PBB = rutin, BPHTB = sekali

## Soal Konseptual
1. BPHTB = pajak atas perolehan hak tanah/bangunan. One-off tax, dikelola daerah, dibayar sebelum balik nama.
2. PBB = pajak kepemilikan (tahunan). BPHTB = pajak transaksi (sekali).
3. Jual beli, hibah, warisan, tukar menukar, lelang, pemasukan ke badan usaha.
4. Kantor pemerintah, kepentingan umum, lembaga internasional, diplomat, konversi hak, wakaf, ibadah, masyarakat berpenghasilan rendah.
5. NPOP = nilai perolehan objek pajak (harga transaksi). NPOPTKP = batas NPOP tidak kena pajak.
6. **5%** dari NPOP Kena Pajak (NPOP - NPOPTKP).
7. Hitung BPHTB -> bayar ke kas daerah -> validasi -> proses balik nama di PPAT/BPN.
8. Sertifikat tidak bisa dibalik nama, terkena sanksi administrasi.

## Soal Aplikasi
1. NPOP Kena Pajak = Rp800jt - Rp80jt = Rp720jt. BPHTB = 5% x Rp720jt = **Rp36.000.000**
2. **Pembeli/penerima hak** yang wajib bayar BPHTB.
3. NPOP Kena Pajak = Rp2M - Rp60jt = Rp1.940.000.000. BPHTB = 5% x Rp1.940.000.000 = **Rp97.000.000**
4. Jual beli NPOPTKP Rp60-80jt. Warisan NPOPTKP Rp300jt. Warisan lebih ringan.
5. PPAT memastikan BPHTB sudah dibayar sebelum AJB ditandatangani.
