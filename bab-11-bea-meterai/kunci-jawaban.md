# KUNCI JAWABAN - BAB 11: Bea Meterai

## Studi Kasus

### Kasus 1: Identifikasi Objek Meterai
1. Ya, `kuitansi-meterai-10jt.pdf` kena Bea Meterai karena nilai > Rp5jt. Di dokumen tertera kotak meterai Rp10.000.
2. `kuitansi-non-meterai-3jt.pdf` tidak kena karena nilai Rp3jt < Rp5jt. Batas objek Bea Meterai adalah > Rp5jt.
3. Ya, surat perjanjian adalah objek Bea Meterai (dokumen perdata). Tarif Rp10.000.
4. Objek: surat perjanjian, akta notaris, kuitansi > Rp5jt. Bukan objek: ijazah, tanda terima gaji, kuitansi pajak.

### Kasus 2: Tarif dan Pelunasan
1. **Rp10.000 per dokumen** (berlaku sejak 1 Jan 2021).
2. Meterai tempel Rp10.000 atau e-Meterai.
3. Meterai tempel = fisik, ditempel. e-Meterai = digital, untuk dokumen elektronik.
4. Dokumen tidak bermeterai tidak dapat diterima sebagai alat bukti di pengadilan.

### Kasus 3: Praktik
1. Kuitansi Rp7jt -> **Objek** (> Rp5jt)
2. Kuitansi Rp4jt -> **Bukan objek** (< Rp5jt)
3. Tanda terima gaji -> **Bukan objek**
4. Akta notaris -> **Objek**
5. Bukti setoran pajak -> **Bukan objek**

## Soal Konseptual
1. Dikenakan atas dokumen (bukan penghasilan/barang). Tarif tetap Rp10.000. Bersifat tidak langsung. Dibayar sekali per dokumen.
2. Surat perjanjian, akta notaris, akta PPAT, surat berharga, dokumen lelang, kuitansi > Rp5jt.
3. Ijazah, tanda terima gaji, kuitansi pajak, surat penyimpanan barang, surat gadai.
4. **Rp10.000 per dokumen** (UU Bea Meterai No. 10/2020).
5. PPh = penghasilan. PPN = konsumsi. Bea Meterai = dokumen. Tarif Bea Meterai tetap, tidak proporsional.
6. Meterai tempel (ditempel di dokumen fisik) dan e-Meterai (untuk dokumen digital).
7. UU mengatur dokumen yang menyebutkan jumlah uang > Rp5jt terutang Bea Meterai.
8. Sebagai alat bukti di pengadilan dan legalitas dokumen.

## Soal Aplikasi
1. Kuitansi Rp7jt (> Rp5jt) -> Kena Bea Meterai Rp10.000.
2. Kuitansi Rp4jt (< Rp5jt) -> Tidak kena Bea Meterai.
3. Surat perjanjian (objek) -> Kena Bea Meterai Rp10.000. Nilai transaksi tidak mempengaruhi tarif (flat).
4. Objek: akta notaris. Bukan objek: kuitansi pajak, ijazah, surat gadai.
5. Dokumen tidak bermeterai tidak memiliki kekuatan hukum sebagai alat bukti.
