# KUNCI JAWABAN - BAB 13: Coretax

## Studi Kasus

### Kasus 1: Pembayaran via ID Billing
1. NPWP, nama WP, jenis pajak, masa pajak, jumlah setor, jatuh tempo, status, NTPN.
2. **026 12345 67890 123**
3. Mendapatkan NTPN (Nomor Transaksi Penerimaan Negara) sebagai bukti pembayaran sah.
4. Sebagai bukti bahwa pembayaran telah diterima negara, digunakan untuk pelaporan SPT.

### Kasus 2: e-Bupot PPh 23
1. Total DPP = Rp325.000.000, Total PPh 23 = **Rp16.900.000**
2. CV Konsultindo Utama (jasa), PT Cipta Karya Inovasi (royalti), PT Alat Berat Nusantara (sewa).
3. Jasa konsultan, royalti software, sewa alat berat.
4. Hemat waktu, mengurangi kesalahan, terintegrasi dengan SPT, data tersimpan otomatis.

### Kasus 3: Rekonsiliasi e-Faktur
1. PK = Rp16.500.000 + Rp8.250.000 + Rp5.500.000 = **Rp30.250.000**. PM = **Rp11.000.000**
2. PPN Kurang Bayar = Rp30.250.000 - Rp11.000.000 = **Rp19.250.000**
3. Sesuai dengan perhitungan di SPT Masa PPN.

### Kasus 4: Pelaporan SPT Elektronik
1. Nomor transaksi, tgl lapor, NPWP, nama, jenis SPT, masa pajak, status SPT.
2. Untuk verifikasi keaslian dokumen SPT elektronik.
3. Cepat, murah, akurat, langsung terverifikasi, mengurangi kontak fisik, ramah lingkungan.

## Soal Konseptual
1. Sistem administrasi perpajakan digital terintegrasi DJP.
2. Sebagai kode unik untuk pembayaran pajak melalui bank/pos persepsi.
3. e-Bupot = bukti potong PPh 23/26. e-Faktur = faktur pajak PPN.
4. Lebih efisien, akurat, dan terintegrasi langsung dengan SPT.
5. Buat ID Billing -> bayar ke bank/pos -> dapat NTPN -> laporkan di SPT.
6. Sebagai bukti lapor dan arsip digital.
7. Coretax memungkinkan DJP merekonsiliasi data pelaporan WP secara real-time.
8. PKP bisa mengkreditkan PM hanya jika memiliki e-Faktur yang valid.

## Soal Aplikasi
1. ID Billing untuk PPh 23 Rp16.900.000: kode 411124, masa pajak sesuai, dibayar sebelum jatuh tempo.
2. Total = Rp4.000.000 + Rp12.000.000 + Rp900.000 = **Rp16.900.000**
3. PPN disetor = **Rp19.250.000**
4. Login DJP Online -> pilih SPT -> isi data -> hitung -> upload lampiran -> kirim -> dapat BPS.
5. Sebagai bukti hukum SPT telah diterima DJP dan arsip untuk keperluan perpajakan.
