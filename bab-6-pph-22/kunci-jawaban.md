# KUNCI JAWABAN - BAB 6: PPh Pasal 22

## Studi Kasus

### Kasus 1: Impor Barang
1. CIF = USD 350.000 + 3.500 + 7.500 = **USD 361.000**
   CIF (Rp) = 361.000 x Rp15.500 = **Rp5.595.500.000**
2. Nilai Impor = Rp5.595.500.000 + 7,5% x Rp5.595.500.000 = Rp5.595.500.000 + Rp419.662.500 = **Rp6.015.162.500**
3. PPh 22 = 2,5% x Rp6.015.162.500 = **Rp150.379.062**
4. Dibayar bersamaan dengan bea masuk sebelum barang dikeluarkan dari pelabuhan.

### Kasus 2: Pembelian Pemerintah
1. Ya, pembelian oleh bendahara pemerintah adalah objek PPh 22.
2. **Bendahara Kementerian PUPR** sebagai pemungut.
3. PPh 22 = 1,5% x Rp200.000.000 = **Rp3.000.000**
4. Diterima = Rp200.000.000 - Rp3.000.000 = **Rp197.000.000**

### Kasus 3: Identifikasi Tarif
1. Impor tanpa API: **7,5%** x Rp1M = Rp75jt
2. Penjualan semen: **0,25%** x Rp500jt = Rp1.250.000
3. Penjualan mobil ATPM: **0,45%** x (10 x Rp400jt) = 0,45% x Rp4M = **Rp18.000.000**
4. BBM dari Pertamina ke SPBU: **0,25%**
5. Ekspor batubara: **1,5%** x Rp5M = Rp75jt

## Soal Konseptual
1. PPh 22 = barang & perdagangan (dipungut). PPh 21 = tenaga kerja (dipotong). PPh 23 = jasa & modal (dipotong).
2. Bank Devisa/DJBC (impor), bendahara pemerintah, BUMN, industri tertentu (semen, kertas, baja, otomotif, farmasi), ATPM.
3. Dibayar di muka dalam rantai distribusi, nanti dikreditkan di SPT Tahunan sebagai kredit pajak.
4. Impor API (2,5%), tanpa API (7,5%), bendahara (1,5%), semen (0,25%), mobil (0,45%), ekspor tambang (1,5%).
5. Dipotong = pajak diambil dari pembayaran ke penerima. Dipungut = pemungut menarik pajak dari pembeli.
6. PPh 22 menjadi kredit pajak yang mengurangi PPh Badan terutang di SPT Tahunan.
7. Barang dengan harga sangat tinggi: rumah > Rp30M, pesawat pribadi, yacht, mobil > Rp2M, motor > Rp300jt.
8. Final hanya untuk penjualan barang mewah tertentu. Sisanya tidak final.
