# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

**BrevetAB-VirtualLab** — Indonesian Tax (Brevet Pajak A–B) course materials. A self-paced learning program covering Indonesian taxation from fundamentals to integrated case studies. 14 chapters total.

## Repository Structure

```
├── Sylabus.html                       # Course syllabus & orientation (Bab 0)
├── MATERI BREVET PAJAK AB-fixed1.pdf  # Full course material (~2 MB, 14 chapters)
└── CLAUDE.md                          # This file
```

- **`Sylabus.html`** — Standalone HTML file (inline styles, no framework). Bab 0 orientation: course structure, learning flow, grading system, etiquette, pre-test topics.
- **`MATERI BREVET PAJAK AB-fixed1.pdf`** — Complete course material. Chapters 1–14 covering: KUP, PPh OP, PPh Badan, PPh 21/22/23/26, PPh Final 4(2), PPN & PPnBM, PBB, Bea Meterai, BPHTB, Coretax, and Integrative Case Study.

## Course Chapters

| Ch | Topic | Focus |
|---|---|---|
| 1 | KUP | Tax basics, NPWP, SPT, taxpayer rights/obligations |
| 2 | PPh OP | PTKP, PKP, progressive rates, individual calc |
| 3 | PPh Badan | Fiscal profit/loss, fiscal correction, corporate rates, MSME |
| 4 | PPh 21 | Salary, honorarium, employees, non-employees, withholding slips |
| 5 | PPh 23 | Services, royalties, interest, dividends, non-land/building rent |
| 6 | PPh 22 | Import, govt procurement, SOE transactions |
| 7 | PPh 26 | Foreign taxpayer income, tax treaty |
| 8 | PPh Final 4(2) | Land/building rent, MSME final, construction, land/building transfer |
| 9 | PPN & PPnBM | BKP, JKP, PKP, Input/Output Tax, tax invoices |
| 10 | PBB | NJOP, NJOPTKP, NJKP, SPPT, land/building tax calc |
| 11 | Bea Meterai | Document tax, physical & e-Meterai |
| 12 | BPHTB | Land/building acquisition, NPOP, tax calc |
| 13 | Coretax | Digital tax admin, billing, e-Bupot, e-Faktur, reconciliation |
| 14 | Studi Kasus Terpadu | Company tax simulation & capstone project |

## Grading System

- Kuis Formatif: 25%
- Tugas Mandiri: 25%
- Forum Diskusi: 15%
- Studi Kasus Akhir: 25%
- Post-Test: 10%

## Development

This is an educational content repository — no build system, package manager, or test framework. The HTML syllabus uses inline styles with no external dependencies and can be opened directly in any browser.
