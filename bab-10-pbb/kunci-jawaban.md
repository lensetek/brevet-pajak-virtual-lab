# KUNCI JAWABAN - BAB 10: Pajak Bumi dan Bangunan (PBB)

## Studi Kasus

### Kasus 1: Hitung PBB Perusahaan
1. Luas tanah: 500 m2, Luas bangunan: 400 m2
2. NJOP Bumi = 500 x Rp2.500.000 = Rp1.250.000.000. NJOP Bangunan = 400 x Rp1.500.000 = Rp600.000.000. Total NJOP = **Rp1.850.000.000**
3. NJOPTKP = **Rp12.000.000**
4. NJOP untuk PBB = Rp1.850.000.000 - Rp12.000.000 = Rp1.838.000.000. NJKP = 20% x Rp1.838.000.000 = **Rp367.600.000**
5. PBB = 0,1% x Rp367.600.000 = **Rp367.600** (cocok dengan `sppt-pbb.pdf`)
6. Paling lambat 6 bulan sejak SPPT diterima.

### Kasus 2: Perbandingan NJOP
1. Menteng (Rp5jt/m2 bumi, Rp3jt/m2 bangunan)
2. NJOP Bumi = 200 x Rp5jt = Rp1.000.000.000
3. Lokasi strategis, akses transportasi, fasilitas umum, kondisi lingkungan.
4. Semakin tinggi NJOP, semakin besar PBB terutang.

## Soal Konseptual
1. PBB = pajak atas keberadaan bumi/bangunan. Karakteristik: kebendaan (object-based tax), tahunan, dikelola daerah.
2. NJOP = nilai jual objek pajak. NJOPTKP = batas NJOP tidak kena pajak. NJKP = persentase NJOP yang kena pajak.
3. PPh = penghasilan. PPN = konsumsi. PBB = aset/kepemilikan.
4. Objek: tanah, rumah, apartemen, gedung, pabrik. Bukan objek: hutan lindung, ibadah, pendidikan, kuburan.
5. Lokasi, luas, kondisi, fasilitas, NJOP pasar.
6. PBB = Tarif (0,5%) x NJKP. NJKP = % x (NJOP - NJOPTKP).
7. UU No. 28/2009 tentang PDRD memberikan kewenangan pemungutan PBB-P2 ke daerah.
8. Orang pribadi/badan yang memiliki/menguasai/memanfaatkan objek pajak.

## Soal Aplikasi
1. Total NJOP = (200x2jt) + (150x1jt) = Rp400jt + Rp150jt = Rp550jt. NJOP untuk PBB = Rp550jt - Rp12jt = Rp538jt. NJKP = 20% x Rp538jt = Rp107,6jt. PBB = 0,1% x Rp107,6jt = **Rp107.600**
2. NJOP untuk PBB = Rp550jt - Rp20jt = Rp530jt. NJKP = 20% x Rp530jt = Rp106jt. PBB = 0,1% x Rp106jt = **Rp106.000**
3. NJOP = (500x2,5jt) + (300x1,5jt) = Rp1.250jt + Rp450jt = Rp1.700jt. NJOP untuk PBB = Rp1.700jt - Rp12jt = Rp1.688jt. NJKP = 20% x Rp1.688jt = Rp337,6jt. PBB = 0,1% x Rp337,6jt = **Rp337.600**
4. Lokasi strategis, NJOP berbeda per zona, ada fasilitas tambahan.
5. NTPN, tanggal bayar, masa pajak, jumlah bayar, bank persepsi.
