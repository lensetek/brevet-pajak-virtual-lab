# KUNCI JAWABAN - BAB 4: PPh Pasal 21

## Studi Kasus

### Kasus 1: PT Maju - 5 Karyawan

1. **Penghasilan Bruto per Bulan:**
   - Budi: Rp8jt + Rp2,75jt = **Rp10.750.000**
   - Siti: Rp4,5jt + Rp550rb = **Rp5.050.000**
   - Doni: Rp15jt + Rp4,75jt = **Rp19.750.000**
   - Rina: Rp6jt + Rp1,2jt = **Rp7.200.000**
   - Ahmad: Rp10jt + Rp2,5jt = **Rp12.500.000**

2. **Take Home Pay:**
   - Budi: Rp10.750.000 - Rp200rb - Rp150rb - Rp248.500 = **Rp10.151.500**
   - Siti: Rp5.050.000 - Rp105rb - Rp23.750 = **Rp4.921.250**
   - Doni: Rp19.750.000 - Rp240rb - Rp200rb - Rp635.000 = **Rp18.675.000**
   - Rina: Rp7.200.000 - Rp150rb - Rp100rb - Rp82.500 = **Rp6.867.500**
   - Ahmad: Rp12.500.000 - Rp240rb - Rp150rb - Rp350.000 = **Rp11.760.000**

3. **PPh 21 terbesar:** Doni (Rp635.000) karena gaji tertinggi, status K/2.

4. **Total PPh 21 disetor:** Rp248.500 + Rp23.750 + Rp635.000 + Rp82.500 + Rp350.000 = **Rp1.339.750/bulan**

### Kasus 2: Pegawai Tidak Tetap
- Penghasilan per hari: Rp500.000
- PTKP per hari (TK/0): Rp54.000.000 / 360 = Rp150.000
- Hari ke-1 s.d. ke-17: penghasilan kumulatif belum > Rp4.500.000, PPh 21 = 0
- Hari ke-18: total penghasilan = 18 x Rp500.000 = Rp9.000.000
- PKP = Rp9.000.000 - (18 x Rp150.000) = Rp9.000.000 - Rp2.700.000 = Rp6.300.000
- PPh 21 hari ke-18 = 5% x Rp6.300.000 = **Rp315.000**

### Kasus 3: Bukan Pegawai (Konsultan)
1. DPP = 50% x Rp17.000.000 = **Rp8.500.000**
2. PPh 21 = 5% x Rp8.500.000 = **Rp425.000**
3. Diterima = Rp17.000.000 - Rp425.000 = **Rp16.575.000**

### Kasus 4: Perbandingan
- **Pegawai tetap:** penghasilan teratur bulanan, potong PPh 21 tiap bulan menggunakan TER/SPT tahunan
- **Pegawai tidak tetap:** upah harian/mingguan, potong saat penghasilan > Rp4.500.000 kumulatif
- **Bukan pegawai:** DPP 50% x penghasilan bruto, tarif progresif Pasal 17

## Soal Konseptual

1. **PPh 21** = pajak dipotong pemberi kerja (withholding). **PPh OP** = pajak dihitung sendiri di SPT Tahunan. PPh 21 adalah kredit pajak untuk PPh OP.
2. Withholding tax memudahkan pemungutan, mengurangi risiko penghindaran, dan memberi kepastian penerimaan negara.
3. Pemberi kerja, bendahara pemerintah, dana pensiun, penyelenggara kegiatan.
4. Pegawai tetap: penghasilan bruto - biaya jabatan - iuran pensiun. Pegawai tidak tetap: PTKP harian. Bukan pegawai: DPP 50%.
5. Biaya jabatan = 5% dari penghasilan bruto, maks Rp500.000/bulan atau Rp6.000.000/tahun.
6. TER adalah tarif efektif rata-rata untuk menyederhanakan perhitungan PPh 21 bulanan.
7. Gaji, tunjangan, THR/bonus, upah lembur, honorarium.
8. Natura objek pajak: rumah dinas di luar daerah, kendaraan dinas untuk pejabat, fasilitas golf. Natura dikecualikan: konsumsi kantor, fasilitas kesehatan BPJS, sarana kerja.

## Soal Aplikasi

1. **Karyawan Rp8jt, K/1:**
   - Bruto setahun: Rp10.750.000 x 12 = Rp129.000.000
   - Biaya jabatan: 5% x Rp129jt = Rp6.000.000 (maks)
   - Iuran pensiun: Rp150rb x 12 = Rp1.800.000
   - Neto: Rp129jt - Rp6jt - Rp1,8jt = Rp121.200.000
   - PTKP K/1: Rp63.000.000
   - PKP: Rp58.200.000
   - PPh 21: 5% x Rp58.200.000 = **Rp2.910.000/tahun** = **Rp242.500/bulan**

2. **Pegawai tidak tetap 18 hari, Rp400rb/hari:**
   - Total: 18 x Rp400rb = Rp7.200.000
   - PTKP: 18 x (Rp54jt/360) = 18 x Rp150rb = Rp2.700.000
   - PKP: Rp7.200.000 - Rp2.700.000 = Rp4.500.000
   - PPh 21 = 5% x Rp4.500.000 = **Rp225.000**

3. **Konsultan Rp25jt:** DPP = 50% x Rp25jt = Rp12.500.000. PPh 21 = 5% x Rp12.500.000 = **Rp625.000**

4. Gaji Rp5jt/bulan, TK/0. Bruto setahun = Rp60jt. Biaya jabatan Rp3jt. Neto = Rp57jt. PTKP = Rp54jt. PKP = Rp3jt. PPh 21 = 5% x Rp3jt = **Rp150.000/tahun** = **Rp12.500/bulan**. TER mungkin memberikan hasil berbeda karena menggunakan pendekatan rata-rata.

5. THR digabung dengan penghasilan bulan Juni, dihitung PPh 21-nya dengan menjadikan penghasilan setahun disetahunkan, selisihnya adalah pajak atas THR.
